## 4. מגבלות הסריקה והשלמה ידנית

1. **מה נסרק:** ACODE1–ACODE9, ACODE4_MAP, TOOLS_MAP, COURSE_FLOWCHART (קבצי `CODE*.HTML` הישנים הוחרגו — גרסאות קודמות).
2. **מה לא בוצע (חסימת רשת):** אימות כותרות/זמינות ב-oEmbed; זיהוי ערוץ ("שלה"/חיצוני); שליפת רשימת הערוץ "חולמים תקשוב"; הצלבה והצעות שיבוץ.
3. **להשלמה — הריצו בפווארשל (מהתיקייה של הפרויקט):**

```powershell
# א. אימות כל המזהים מול oEmbed ועדכון ה-JSON
$inv = Get-Content .\video-inventory.json -Raw -Encoding UTF8 | ConvertFrom-Json
$ids = $inv.embedded.videoId | Where-Object { $_ -match '^[A-Za-z0-9_-]{11}$' } | Sort-Object -Unique
$results = foreach ($id in $ids) {
  try {
    $r = Invoke-RestMethod "https://www.youtube.com/oembed?url=https://www.youtube.com/watch?v=$id&format=json"
    [pscustomobject]@{ videoId=$id; title=$r.title; channel=$r.author_name; status='ok' }
  } catch { [pscustomobject]@{ videoId=$id; title=$null; channel=$null; status='unavailable' } }
  Start-Sleep -Milliseconds 200
}
$results | ConvertTo-Json | Out-File .\oembed-results.json -Encoding utf8
$results | Where-Object status -eq 'unavailable'   # <- סרטונים שבורים

# ב. רשימת הערוץ: מצאו את ה-handle (חיפוש "חולמים תקשוב" ביוטיוב), ואז:
$h = @{ 'User-Agent'='Mozilla/5.0'; 'Accept-Language'='he' }
$html = (Invoke-WebRequest "https://www.youtube.com/@HANDLE/videos" -Headers $h).Content
$channelId = [regex]::Match($html,'"channelId":"(UC[^"]+)"').Groups[1].Value
[xml]$rss = (Invoke-WebRequest "https://www.youtube.com/feeds/videos.xml?channel_id=$channelId" -Headers $h).Content
$rss.feed.entry | ForEach-Object { "{0}`t{1}`t{2}" -f $_.videoId, $_.published, $_.title }
```

4. גם לאחר ההשלמה, דף `/videos` ו-RSS מציגים רק את ~30/15 הסרטונים האחרונים; לכיסוי מלא יש לעבור גם על הפלייליסטים של הערוץ.
