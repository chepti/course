# הבית של חופית — דף בית 360

פורטפוליו אינטראקטיבי: חללי 360 עם נקודות לחיצה (hotspots).

- **כתובת:** https://chepti.com (דף הבית) — הקבצים יושבים ב-`/home/` ומוגשים בשורש דרך rewrite.
- **קבצי המקור:** `site/` — נפרסים ל־`~/public_html/home/` בהוסטינגר.

## דף הבית (שורש chepti.com)

השורש מגיש את העמוד דרך כלל ב-`~/public_html/.htaccess` (בלוק `# BEGIN HOME360`):
`RewriteRule ^$ /home/index.html [L]`. כדי שהנכסים ייטענו נכון מהשורש, ל-`index.html` יש `<base href="/home/">`.
גיבוי לפני השינוי: `~/public_html/.htaccess.bak-home360`. לשחזור — למחוק את הבלוק או להעתיק חזרה מהגיבוי.
עמודי וורדפרס האחרים (‎/wilk, ‎/wp-admin וכו') ממשיכים לעבוד כרגיל — רק השורש הריק נתפס.

הקישור ליצירת קשר נכפה בקוד לוואטסאפ (`DEFAULT_CONTACT`) כל עוד `settings.contactUrl` אינו קישור http(s) — כי אין עורך הגדרות בפאנל ושמירות של אדמין עלולות להחזיר ערך ישן.

`site/home.htaccess` נפרס כ-`~/public_html/home/.htaccess` — מונע cache של קבצי ‎.html (no-cache) כדי שעדכוני קוד יופיעו מיד בלי צורך ב-hard refresh.

## מודל הנקודות (סכימה)

לכל נקודה יש `kind`:

- **card** (ברירת מחדל) — כרטיס מורכב שנפתח בלחיצה. **לעולם לא זורק החוצה אוטומטית.** מרכיב יחד רק את מה שמולא:
  - `category` — אחת מ: pwa / appsscript / tool / portfolio / game / content / family → תגית צבעונית + קיבוץ ברשימה.
  - `image` — כתובת URL **או** תמונה מודבקת / נגררת / קובץ. מודבקת נשמרת כ-data URL ב-`data.json` (מוקטנת אוטומטית ל-1280px JPEG). בכרטיס התמונה **מוכלת** (object-fit: contain, לא נחתכת) ומוצגת בצד אחד, הטקסט בצד השני (נערם במובייל). הדבקה: ממקדים את מסגרת ההדבקה או כל מקום בפאנל ומקישים Ctrl+V (מאזין paste ברמת document); לבחירת קובץ יש כפתור נפרד "בחרי קובץ".
  - `description`, `embed` (iframe בתוך הכרטיס), `video` (יוטיוב/MP4), `lottie`.
  - `link` + `linkLabel` — כפתור CTA שפותח את הפריט בטאב חדש.
- **room** — דלת. `target` = מזהה החלל. מעבר חלק עם פייד, בלי חלונית.
- **videowall** — `video` (MP4) מוקרן על משטח בתוך הסצנה (מושתק, בלופ). לחיצה פותחת בגדול.

כל נקודה כוללת גם:

- `tier` — היררכיה ויזואלית: `title` (כותרת אזור — טקסט מודגש בלי עיגול אימוג'י), `normal` (עיגול רגיל), `small` (עיגול קטן). כותרת בלי תוכן לא פותחת כלום; כותרות מוסתרות מרשימת "כל הקישורים".
- `stats` — מערך של `{value, label}` שמוצג ככרטיסי מספרים גדולים בתוך הכרטיס (למשל 250 / סרטונים).
- `date` — תאריך חופשי (טקסט) שמוצג ליד התגית בכרטיס.

## ניהול אדמין נוסף

- **העברת נקודה למרחב אחר** — שדה "מרחב" בפאנל העריכה; שינוי מעביר את הנקודה לחלל היעד.
- **מרחבים** — כפתור "מרחב" (עריכת שם + תמונת רקע 360° של החלל הנוכחי) וכפתור "מרחב חדש". תמונת הרקע נשמרת כ-data URL (מוקטן ל-4096px) ומתחלפת בזמן אמת. מחיקת מרחב מנקה דלתות שהצביעו אליו.
- כפתורי מחיקה/סיום צפים בראש הפאנל (sticky). אזור התמונה קומפקטי — הדבקה/גרירה גלויה, "עוד אפשרויות" חושף בחירת קובץ + הדבקת URL.

הפונקציה `normalizeHotspot()` ממירה סכימה ישנה (`type`/`content`) לחדשה בטעינה ומוסיפה ברירות מחדל ל-`tier`/`stats`, כך שנתונים ישנים ממשיכים לעבוד.

## טכני

- מנוע: Three.js r128, ספרה עם טקסטורת equirectangular.
- מיפוי קואורדינטות: `lon = xfrac * 360` (שמאל התמונה = 0°, מרכז = 180°), `lat` חיובי = למעלה.
- `api.php` טוען את `wp-load.php` מהתיקייה שמעליו לאימות אדמין + nonce. זהירות: אין להשתמש במשתנה בשם `$action` לפני טעינת וורדפרס — תוספים דורסים אותו.
- שמירה יוצרת גיבוי אוטומטי `data.backup.json` בשרת.
- קישור ישיר לחלל: `?space=learning`.
