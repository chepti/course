## מודל הנקודות (סכימה)

לכל נקודה יש `kind`:

- **card** (ברירת מחדל) — כרטיס מורכב שנפתח בלחיצה. **לעולם לא זורק החוצה אוטומטית.** מרכיב יחד רק את מה שמולא:
  - `category` — אחת מ: pwa / appsscript / tool / portfolio / game / content / family → תגית צבעונית + קיבוץ ברשימה.
  - `image` — כתובת URL **או** תמונה מודבקת (Ctrl+V) / נגררת / קובץ. מודבקת נשמרת כ-data URL ב-`data.json` (מוקטנת אוטומטית ל-1280px JPEG). אין צורך להעלות ידנית.
  - `description`, `embed` (iframe בתוך הכרטיס), `video` (יוטיוב/MP4), `lottie`.
  - `link` + `linkLabel` — כפתור CTA שפותח את הפריט בטאב חדש.
- **room** — דלת. `target` = מזהה החלל. מעבר חלק עם פייד, בלי חלונית.
- **videowall** — `video` (MP4) מוקרן על משטח בתוך הסצנה (מושתק, בלופ). לחיצה פותחת בגדול.

כל נקודה כוללת גם:

- `tier` — היררכיה ויזואלית: `title` (כותרת אזור — טקסט מודגש בלי עיגול אימוג'י), `normal` (עיגול רגיל), `small` (עיגול קטן). כותרת בלי תוכן לא פותחת כלום; כותרות מוסתרות מרשימת "כל הקישורים".
- `stats` — מערך של `{value, label}` שמוצג ככרטיסי מספרים גדולים בתוך הכרטיס (למשל 250 / סרטונים).

הפונקציה `normalizeHotspot()` ממירה סכימה ישנה (`type`/`content`) לחדשה בטעינה ומוסיפה ברירות מחדל ל-`tier`/`stats`, כך שנתונים ישנים ממשיכים לעבוד.

## טכני

- מנוע: Three.js r128, ספרה עם טקסטורת equirectangular.
- מיפוי קואורדינטות: `lon = xfrac * 360` (שמאל התמונה = 0°, מרכז = 180°), `lat` חיובי = למעלה.
- `api.php` טוען את `wp-load.php` מהתיקייה שמעליו לאימות אדמין + nonce. זהירות: אין להשתמש במשתנה בשם `$action` לפני טעינת וורדפרס — תוספים דורסים אותו.
- שמירה יוצרת גיבוי אוטומטי `data.backup.json` בשרת.
- קישור ישיר לחלל: `?space=learning`.
