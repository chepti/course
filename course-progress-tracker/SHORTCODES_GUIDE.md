# מדריך לשימוש בשורטקודים

## שורטקודים זמינים

### 1. שורטקוד כללי - כל היחידות
הצג את ההתקדמות בכל היחידות:

```
[user_course_progress]
```

**שימוש:**
- הוסף את השורטקוד לכל עמוד או פוסט
- למשל, בעמוד פרופיל אישי או בעמוד דף הבית

**דוגמה:**
```
<h2>ההתקדמות שלי בקורס</h2>
[user_course_progress]
```

---

### 2. שורטקוד ספציפי - יחידה אחת
הצג את ההתקדמות ביחידה ספציפית:

```
[unit_progress unit_id="123"]
```

**פרמטרים:**
- `unit_id` - מזהה היחידה (Post ID) - חובה

**איך למצוא את מזהה היחידה:**
1. היכנסי לעמוד היחידה
2. עברי לעריכה (Edit)
3. בשורת הכתובת תראי משהו כמו: `post.php?post=123&action=edit`
4. המספר `123` הוא ה-`unit_id`

**שימוש:**
- הוסף את השורטקוד בעמוד היחידה עצמו
- או בעמוד פרופיל אישי עם כמה יחידות

**דוגמאות:**
```
<!-- יחידה 1 -->
[unit_progress unit_id="32881"]

<!-- יחידה 2 -->
[unit_progress unit_id="32882"]

<!-- יחידה 3 -->
[unit_progress unit_id="32883"]
```

---

## איפה להוסיף את השורטקודים?

### אופציה 1: עמוד פרופיל אישי
אם יש לך עמוד פרופיל אישי, הוסיפי שם:

```
[user_course_progress]
```

### אופציה 2: עמוד נפרד "ההתקדמות שלי"
צרי עמוד חדש בשם "ההתקדמות שלי" והוסיפי:

```
[user_course_progress]
```

### אופציה 3: בעמוד היחידה עצמו
הוסיפי את השורטקוד הספציפי בתחתית עמוד היחידה:

```
[unit_progress unit_id="32881"]
```

### אופציה 4: בסרגל האדמין (אוטומטי)
הקישור "ההתקדמות שלי" מופיע אוטומטית בסרגל השחור למעלה לכל משתמש מחובר.

---

## איך החישוב עובד?

כל יחידה מחולקת ל-4 פרקים עיקריים:
1. **מבט על (overview)** - 25%
2. **כלים (tools)** - 25%
3. **דיון (discussion)** - 25%
4. **משימה (task)** - 25%

כל פרק יכול להיות 0-100% השלמה, והחישוב הסופי הוא:
- אם פרק "מבט על" הושלם ב-100% = 25% מהכלל
- אם פרק "כלים" הושלם ב-50% = 12.5% מהכלל
- אם פרק "דיון" הושלם ב-100% = 25% מהכלל
- אם פרק "משימה" הושלם ב-100% = 25% מהכלל
- **סה"כ: 87.5%**

---

## בעיות נפוצות

### השורטקוד לא מופיע
- ודאי שהמשתמש מחובר
- ודאי שהשורטקוד כתוב נכון (ללא שגיאות הקלדה)
- ודאי שיש פעילות במערכת (אם אין פעילות, יופיע "עוד לא התחלת")

### השורטקוד מציג 0%
- ודאי שיש פעילות במערכת
- בדקי שהמעקב עובד (הודעות ירוקות בעמוד)

### השורטקוד הספציפי לא עובד
- ודאי שה-`unit_id` נכון
- ודאי שהמשתמש התחיל את היחידה הזו
