# פתרון שלב אחר שלב - Course Progress Tracker לא נטען

## הבעיה:
הסקריפט לא נטען - אין הודעות בקונסול ואין הודעות ויזואליות על המסך.

## פתרון שלב אחר שלב:

### שלב 1: בדוק אם אתה רואה הודעה על המסך

**פתח את עמוד הקורס** ו**רענן את העמוד** (F5).

**חפש בפינה הימנית העליונה:**
- ✅ הודעה ירוקה: "Course Progress Tracker פעיל"
- ⚠️ הודעה אדומה: "Course Progress Tracker לא נטען!"

**אם אתה לא רואה כלום** → הסקריפט לא נטען בכלל.

---

### שלב 2: בדוק אם הקוד נוסף ל-functions.php

1. **פתח את הקובץ:**
   ```
   wp-content/themes/astra/functions.php
   ```

2. **חפש בקובץ את הטקסט:**
   ```
   cpt_enqueue_course_scripts
   ```

**אם לא מצאת:**
- הקוד לא נוסף → עבר לשלב 3

**אם מצאת:**
- הקוד קיים → עבר לשלב 4

---

### שלב 3: הוסף את הקוד ל-functions.php

1. **פתח את הקובץ:**
   ```
   wp-content/themes/astra/functions.php
   ```

2. **פתח את הקובץ:**
   ```
   course-progress-tracker/FUNCTIONS_PHP_CODE.txt
   ```

3. **העתק את כל התוכן** מ-`FUNCTIONS_PHP_CODE.txt`

4. **הדבק בסוף הקובץ** `functions.php` (לפני הסוגר `?>` אם יש, או בסוף הקובץ)

5. **שמור את הקובץ**

6. **רענן את עמוד הקורס** (F5)

---

### שלב 4: בדוק את התנאי

הקוד צריך לכלול:
```php
$parent_page = get_page_by_path('aia');
```

**אם ה-slug של עמוד האב שלך הוא לא "aia":**

1. **גלה מה ה-slug הנכון:**
   - היכנס ל-WordPress Admin
   - עבור ל-עמודים → כל העמודים
   - פתח את עמוד האב (העמוד הראשי של הקורס)
   - תחת "קישור קבוע" תראה את ה-slug

2. **שנה את השורה:**
   ```php
   $parent_page = get_page_by_path('aia');
   ```
   ל:
   ```php
   $parent_page = get_page_by_path('הסלאג-הנכון-כאן');
   ```

3. **שמור את הקובץ**

4. **רענן את עמוד הקורס** (F5)

---

### שלב 5: בדוק אם הסקריפט נטען (Network Tab)

1. **פתח את כלי המפתחים** (F12)

2. **לחץ על הכרטיסייה Network** (רשת)

3. **רענן את העמוד** (F5)

4. **חפש בקובץ:**
   ```
   course-tracker.js
   ```

**אם אתה רואה את הקובץ ברשימה:**
- ✅ הסקריפט נטען!

**אם אתה לא רואה את הקובץ:**
- ❌ הסקריפט לא נטען → בדוק את התנאי (שלב 4)

---

### שלב 6: בדוק שאין שגיאות PHP

אם יש שגיאות PHP ב-`functions.php`, התוסף לא יעבוד.

**איך לבדוק:**
1. פתח את הקובץ `functions.php`
2. ודא שכל הסוגריים סגורים נכון
3. ודא שאין שגיאות תחביר

**אם יש שגיאות:**
- WordPress יראה הודעת שגיאה בעמוד
- בדוק את לוגי השגיאות של WordPress

---

## בדיקה מהירה - האם זה עובד?

אחרי שסיימת את כל השלבים:

1. **פתח את עמוד הקורס**
2. **רענן את העמוד** (F5)
3. **חפש בפינה הימנית העליונה:**
   - ✅ "Course Progress Tracker פעיל" (ירוק) → עובד!
   - ⚠️ "Course Progress Tracker לא נטען!" (אדום) → יש בעיה

---

## עדיין לא עובד?

שלח לי:
1. האם הוספת את הקוד ל-`functions.php`? (כן/לא)
2. מה ה-slug של עמוד האב שלך?
3. האם אתה רואה את ההודעה הירוקה או האדומה על המסך?
4. האם אתה רואה את הקובץ `course-tracker.js` ב-Network tab?

---

## טיפים:

- **רענן את העמוד** אחרי כל שינוי ב-`functions.php`
- **נקה את ה-cache** אם יש לך תוסף cache
- **בדוק שהמשתמש מחובר** - התוסף עובד רק למשתמשים מחוברים
- **ודא שהעמוד הנוכחי הוא ילד של עמוד האב** - התנאי בודק את זה
